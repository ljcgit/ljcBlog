## 1.location匹配优先级
+ =：进行普通字符精确匹配，也就是完全匹配
+ ^~：表示普通字符匹配，使用前缀匹配
+ \~\\\~\*：表示执行一个正则匹配，~区分大小写，\~*不区分大小写

## 2.alias和root的使用区别
+ alias在查找文件时直接按照配置给出的路径
+ root需要将配置路径和URL路径结合起来查找

## 3.ab接口压力测试工具
+ 安装 ： yum install httpd-tools
+ 使用
> ab -n 2000 -c 2 http://127.0.0.1/
    -n:总的请求数
    -c:并发数
    -k:是否开启长连接
